# Zweck

Dieses Dokument definiert sämtliche Zauber des Spiels.

Für jeden Zauber werden beschrieben:

- Fantasy
- Hook
- Mechanik
- Timing
- Visualisierung
- Upgradepfad
- Balancing-Rolle

Konkrete Zahlen orientieren sich an:

→ 05_Balance.md

---

# Globale Zauberregeln

Jeder Zauber besitzt:

- genau eine Schule
- genau einen Typ
- genau einen Hook
- eine eindeutige Fantasy
- einen sofortigen Impact
- Rang I–V

---

# Zaubertypen

## Angriff

Direkter Schaden.

Kann zusätzliche Effekte besitzen.

---

## Schutz

Erzeugt Schilde oder defensive Vorteile.

---

## Status

Verändert den Kampf indirekt.

# Upgrade-Regeln

## Rang I

Grundzauber

---

## Rang II

Verbessert Zahlen.

---

## Rang III

Neuer Gameplay-Effekt.

Keine reine Zahlenverbesserung.

---

## Rang IV

Verbessert Zahlen erneut.

---

## Rang V

Vollendung.

Der Zauber erreicht seine endgültige Form.

# Designregeln

Jeder Zauber muss:

✓ Sofort verständlich sein.

✓ Sofort Wirkung zeigen.

✓ Einen klaren Hook besitzen.

✓ Mindestens einen Hybridbuild unterstützen.

✓ Sich visuell von allen anderen unterscheiden.

---

# Verbotene Designs

Keine Zauber dürfen:

- nur Zahlen erhöhen
- ausschließlich Vorbereitung sein
- ohne Gegenleistung alles lösen
- andere Zauber ersetzen

---

# Definition of Done

Ein Zauber gilt als fertig wenn:

✓ Fantasy funktioniert.

✓ Hook funktioniert.

✓ Mechanik funktioniert.

✓ Timing sinnvoll ist.

✓ Visualisierung verständlich ist.

✓ Rang I–V definiert sind.

✓ Balancebudget vergeben wurde.

✓ Hybridmöglichkeiten existieren.
