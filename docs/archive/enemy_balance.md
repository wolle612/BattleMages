# Battlemages – Enemy Balance

Version: 1.0

Status: Final

---

# Balancing-Regeln

Ziel ist nicht, dass jeder Kampf gleich schwer ist.

Jeder Kampf besitzt einen klaren Zweck.

Normale Gegner sollen innerhalb weniger Rotationen besiegt werden.

Elitegegner fordern den Build.

Mini-Boss und Endboss prüfen den gesamten Run.
