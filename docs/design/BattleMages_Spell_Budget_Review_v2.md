## `FAZIT` 

```
Die Architektur des Spellbooks trägt.
```

```
Die wenigen Anpassungen betreffen ausschließlich: - überbudgetierte
Generatoren, - einzelne Motor-Zauber, - einen unterbudgetierten
Verstärker.
```

```
Grundregel:
```

```
Nicht jeder Zauber muss exakt gleich stark sein.
```

```
Build-Enabler dürfen bewusst etwas stärker sein, sofern sie einen neuen
Archetyp eröffnen und nicht universell in jedem Build die beste Wahl
darstellen.
```

```
BattleMages wird über die Stärke kompletter Rotationen balanciert –
nicht über die Stärke einzelner Zauber.
```
